## SATIS-Faction:Smart Academic Tracking and Intervention System for Baliwasan Senior High School - Stand Alone
</hr>
 #### - An academic website for smart academic tracking and intervention exclusive for Baliwasan Senior High School - Stand Alone.

### System Description
    - This system provides way to make the academic standing of a student be visible real-time, enhancing student performance by tracking down their academic standing and system's academic suggestion to further more improve the students academic standing. This system will address the issues with unknowningly failing grades till the end of the term/quarter/academic year and inefficient grade monitoring leading to student's unaware of their grades failing until its too late.

### Features
 ##### * User Authentication(login/register)
 ##### * Grade Monitoring
 ##### * System Notification
 ##### * System Intervention(tier 1 - tier 3 Intervention)
