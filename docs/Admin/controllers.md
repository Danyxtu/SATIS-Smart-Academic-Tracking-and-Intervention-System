## Admin Controller

**TeacherRegistrationContoller**
**UserController**

### TeacherRegistrationContoller

    - Can do:
        * Get all the teachers per department (index()),
        * Approve teachers registration (approve()),
        * and Download Teachers's Documents (downloadDocument()).

### UserController

    - Can do:
        * Display the admin dashboard with overview stats (dashboard()),
        * Display a listing of all users with search, filter, and sort (index()),
        * Show the form to create a new teacher by admin (create()),
        * Store the newly created user in storage (store())
        * Edit the teachers information in their department (edit()),
        * Update the teachers using the update(),
        * Delete the teacher using the destroy().
